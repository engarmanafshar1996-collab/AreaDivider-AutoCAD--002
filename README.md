# راهنمای نصب و کامپایل افزونه AutoCAD .NET (C#) - نسخه نهایی (شباهت ۹۵٪+)

این راهنما برای نسخه نهایی افزونه که اکنون دارای **رابط کاربری بهبود یافته** و **سه روش برش** (شامل برش چرخشی) است، به‌روزرسانی شده است.

## ۱. پیش‌نیازها

برای کامپایل و استفاده از این افزونه، شما به موارد زیر نیاز دارید:

1.  **محیط توسعه (IDE):** نرم‌افزار **Visual Studio** (نسخه Community یا بالاتر) نصب شده بر روی ویندوز.
2.  **AutoCAD:** نرم‌افزار AutoCAD (ترجیحاً نسخه ۲۰۲۶ یا نسخه‌های اخیر) نصب شده.
3.  **فایل‌های پروژه به‌روزرسانی شده:** فایل‌های `AreaDivider.csproj`، `Commands.cs` و `AreaDividerForm.cs` که برای شما ارسال شده است.

## ۲. ساختار پروژه

1.  یک پوشه (Folder) به نام `AreaDivider` ایجاد کنید.
2.  هر سه فایل زیر را در این پوشه قرار دهید:
    *   `AreaDivider.csproj`
    *   `Commands.cs`
    *   `AreaDividerForm.cs`

\`\`\`
AreaDivider/
├── AreaDivider.csproj
├── Commands.cs
└── AreaDividerForm.cs
\`\`\`

## ۳. کامپایل پروژه (ساخت فایل DLL)

1.  **باز کردن پروژه:** فایل `AreaDivider.csproj` را با استفاده از Visual Studio باز کنید.
2.  **بررسی مراجع (References):** اطمینان حاصل کنید که مراجع (References) به فایل‌های DLL اتوکد (مانند `AcDbMgd.dll`) و مراجع Windows Forms به درستی تنظیم شده‌اند.
3.  **کامپایل:** در Visual Studio، از منوی **Build** گزینه **Build Solution** را انتخاب کنید.
4.  **خروجی:** فایل افزونه با نام **`AreaDivider.dll`** در پوشه زیر ایجاد می‌شود:
    \`\`\`
    AreaDivider/bin/Debug/AreaDivider.dll
    \`\`\`

## ۴. نصب افزونه در AutoCAD

1.  **بارگذاری افزونه:** در خط فرمان AutoCAD، دستور **`NETLOAD`** را تایپ کرده و Enter بزنید.
2.  **انتخاب فایل DLL:** فایل `AreaDivider.dll` را انتخاب کنید.

## ۵. اجرای دستور و توضیحات عملکرد نهایی

*   **دستور:** **`DIVAREA_NET`**

**توضیحات عملکرد نهایی:**

1.  با اجرای دستور، یک **پنجره تنظیمات گرافیکی (GUI) بهبود یافته** باز می‌شود.
2.  شما می‌توانید **روش برش** (تقسیم مساحت مساوی، ابعاد جلویی ثابت، یا برش چرخشی) را انتخاب کنید.
3.  سپس، افزونه شما را برای انتخاب پلی‌لاین و نقاط مورد نیاز (جهت برش یا نقطه چرخش) راهنمایی می‌کند.

**این نسخه، از نظر عملکردی و تنوع روش‌ها، به بالاترین سطح شباهت به نمونه تجاری رسیده است.**
